# Exotic Derivatives Pricer

A Monte Carlo simulation framework for pricing exotic options and quantifying model risk.

## Options Implemented
- Asian options (arithmetic and geometric average)
- Barrier options (knock-in and knock-out)
- Lookback options
- Digital/Binary options

## Features
- Monte Carlo pricing engine with variance reduction techniques
- Antithetic variates and control variates for efficiency
- Model risk analysis comparing Black-Scholes vs alternative models
- Convergence analysis and confidence intervals

## Tools
Python, NumPy, pandas, Matplotlib, SciPy
